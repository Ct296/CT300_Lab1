## Giới thiệu

Một ứng dụng máy tính điện tử được xây dựng bằng **HTML, CSS và JavaScript**, mô phỏng máy tính Casio với giao diện phong cách hiện đại.

Dự án hỗ trợ thực hiện các phép toán cơ bản và một số chức năng nâng cao như phần trăm và dấu ngoặc.

## Tính năng chính

- Thực hiện các phép toán:
  - Cộng (+)
  - Trừ (−)
  - Nhân (×)
  - Chia (÷)
  - Phần trăm (%)
- Hỗ trợ dấu ngoặc ( )
- Hiển thị lịch sử phép tính
- Làm tròn số thập phân đến 4 chữ số
- Xử lý lỗi khi nhập sai biểu thức
- Hỗ trợ nhập bằng bàn phím:
  - Enter → Tính kết quả
  - Backspace → Xóa 1 ký tự
  - Escape → Xóa toàn bộ
